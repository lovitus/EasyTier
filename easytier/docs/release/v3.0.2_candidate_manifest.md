## Included behavior

- Adds an isolated, explicit-only `quic-brutal://` overlay using local
  sender-side `tx_bps` configuration.
- Uses ordinary BBR when `tx_bps` is omitted and leaves all existing `quic://`
  configuration and behavior unchanged.
- Reuses existing listener, connector, endpoint, transport selection and failure
  handling without introducing overlay-specific fallback logic.
- Composes Brutal sender pacing with the existing QUIC Stealth socket. Required
  Stealth fails closed; existing unknown-capability fallback preserves the local
  Brutal/BBR sender mode instead of switching congestion control.
- Adds GUI listener configuration on preconfigured port `11013`.
- Upgrades `quinn-udp` to `0.5.15` and correctly handles a short final Stealth
  QUIC GSO segment.
