Commit and tree IDs are deliberately not embedded as literals in this tracked

script prints the exact IDs and compares the bridge tree against `RECOVERY_REF`
(default `codex/v3.0.0-recovery`) at execution time. Those immutable IDs are
recorded with the external workflow/artifact evidence after the bridge is pushed.
