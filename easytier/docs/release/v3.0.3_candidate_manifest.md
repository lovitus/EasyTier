- Uses Mbps-based `tx_mbps` consistently in GUI, CLI, TOML and RPC URLs.
- Preserves `tx_bps` as a read-compatible 3.0.2 alias without changing the
  effective Brutal sender rate.
- Rejects duplicate, mixed, malformed and out-of-range rate parameters.
- Leaves `quic-brutal` transport, Stealth composition, BBR-on-omission,
  protocol priority, fallback and ordinary `quic://` behavior unchanged.
- Adds focused Rust and frontend compatibility tests plus user documentation.
