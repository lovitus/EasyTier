# EasyTier v3.0.4 Release Notes

EasyTier 3.0.4 improves policy controls and managed Geo data in the GUI while
preserving the existing overlay and network lifecycle behavior.

## Changes

- The GUI home page can enable or disable policy routing and open a focused
  Policy YAML editor. Saved policy changes require restarting that network.
- A conservative runtime indicator reports whether the independently owned Leaf
  policy runtime is actually running, rather than only showing configuration.
- A floating start/stop button is available at the lower right and reuses the
  existing actions; the original controls remain unchanged.
- Production WebViews no longer suppress the system context menu, allowing
  Android text selection, copy and paste menus to appear where the platform
  provides them.
- GeoSite, GeoIP and Country MMDB sources can be edited and updated manually.
  The editor displays the saved SHA-256 after successful validation.
- When the server reports the same content length, EasyTier skips downloading
  only if a valid saved index from a previous successful validation exists.
  Missing or corrupt verification state forces a fresh download and parse.
- Download size limits, declared-length checks and format parsing complete before
  a new rule-data file becomes active.

## 中文摘要

- GUI 首页新增 Policy Routing 启停选择和 Policy YAML 快速编辑；保存后需要
  重启对应网络才会生效。
- 状态栏显示独立 Leaf policy runtime 的真实 Running/Stopped 状态。
- 右下角新增复用原逻辑的悬浮启动/停止按钮，原有按钮继续保留。
- Android WebView 恢复系统文本选择、复制和粘贴菜单。
- GeoSite、GeoIP 和 Country MMDB 均可修改下载地址、手动更新并显示成功
  校验后保存的 SHA-256。
- 只有远端大小相同且存在先前成功验证的有效索引时才跳过下载；缓存缺失或
  损坏会重新下载、校验长度并解析，失败文件不会替换当前数据。
